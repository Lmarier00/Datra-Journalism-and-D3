Poverty vs. Obesity

  • The data set included with this analysis is based on 2014 American Community Survey (ACS) 1-year estimates. The data set includes
  • data on rates of income, obesity, poverty, etc. by state. The data showed no correlation between obesity and poverty. For example,
  • he poverty rate in New Hampshire is approximately 9.2% and the obesity rate is 27.4%. Contrast that state's statistic with
  • New Mexico's similar obesity rate of 28.4% but a much greater poverty rate of 21.3%.
